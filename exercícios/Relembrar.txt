> Conteúdo em HTML
    > Usa tags <>
        > Exemplos:
            > <h1>exemplo de título</h1>
                > <h1>: Abertura de tag
                > </h1>: Fechamento de tag
                > exemplo de título: Conteúdo

            > <img src=''foto.png'' alt=''exemplo de foto''>
                > <img: Abertura de tag
                > alt: parâmetro
                > exemplo de foto: valor

> Estilo em CSS
    > Exemplos:
        h1 {
            font-family: Arial;
            font-size: 20pt;
            color: blue;
        }
            > h1: Seletor
            > {}: Bloco
            > color: propriedade
            > blue: valor

> Códigos
    > <hr>: Horizontal row, faz uma linha na Horizontal.
    > <br>: Break row, quebra a linha.
    > &lt; e &gt;: Digita < e >, respectivamente.
    > <!-- -->: Comentário.
    > link:favicon: Para colocar o favicon.
    > <strong>: Negrito / Destaque.
    > <em>: Itálico / Ênfase.
    > <mark>: Marca o texto.
    > Pode usar style dentro das tags. Ex: <mark style="background-color: lime;">
    > TAG OBSOLETA <big>: Aumenta o tamanho da palavra.
    > <small>: Diminui o tamanho da palavra.
    > <del>: Texto riscado.
    > <ins>: Texto inserido / sublinhado.
    > <sup>: Texto sobrescrito.
    > <sub>: Texto subscrito.
    > <code>: Texto em código fonte / para ler código / Mono espaçada, mesmo espaçamento de letras.
    > <pre>: Pré formatação / Considera quebras de linhas e espaçamentos dentro da tag.
    > <q>: Citação / Coloca aspas.
    > <blockquote>: Citação completa / pula uma linha e da espaçamento. Pode usar parâmetro. Ex: <blockquote cite=''link da citação''>
    > <abbr title=''''>: Significado da abreviação que vc usou.
    > <address>: Texto que é um endereço.
    > <ol>: Ordered list, usado para fazer lista. Exemplos de usar. <ol type=""A""> <li>Acordar</li> </ol> / pode usar parâmetros
    > <ul>: Unordered list, usado para fazer lista. Exemplos de usar. <ul type=""circle""> <li>Acordar</li> </ul> / pode usar parâmetros
    > <a>: âncora. Para colocar links. abrir em outra janela <a target="_blank"> / <a download> e <a type="application/zip"> ou o outro tipo de media type.
